# Voting Platform - Корпоративная система голосования

## Описание проекта
Это **учебный проект** системы голосования для корпоративных опросов, оценки проектов и сбора мнений сотрудников.

## Назначение
Система предназначена для:
- Организации корпоративных опросов и голосований
- Голосования за лучшие проекты/предложения
- Сбора обратной связи от участников
- Подсчета результатов опросов

## Структура базы данных

### Основные сущности:

**1. Polls (Опросы)**
- Хранит информацию о проводимых опросах
- Название, даты начала/окончания, статус

**2. Participants (Участники)**
- Пользователи системы, имеющие право голоса
- ФИО, контактные данные, дата регистрации

**3. Voting (Голосование)**
- Записи о поданных голосах
- Связь участника с опросом и выбранным вариантом

**4. Options (Варианты)**
- Доступные варианты ответа в опросе
- Описание вариантов

**5. Teams (Команды/Группы)**
- Группировка участников по командам
- Дата регистрации команды

**6. Results (Результаты)**
- Итоги голосования
- Количество голосов, процент, дата обработки

**7. Feedback (Обратная связь)**
- Отзывы и предложения участников
- Описание, дата создания, статус

**8. Roles & Permissions (Роли и права доступа)**
- Управление правами пользователей
- Роли: администратор, модератор, участник

## Технологии
- База данных: реляционная СУБД
- Схема: нормализованная структура

## Использование
Проект создан в **образовательных целях** для демонстрации:
- Проектирования баз данных
- Организации систем голосования
- Работы с пользовательскими правами

## Примеры использования
- Голосование за лучший проект хакатона
- Опросы сотрудников компании
- Выбор названия для нового продукта
- Оценка предложений по улучшению процессов
