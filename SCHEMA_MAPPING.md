# Маппинг терминологии базы данных

Этот документ описывает соответствие терминов в адаптированной системе голосования.

## Переименование таблиц

| Исходное название (RU) | Новое название (EN) | Назначение |
|------------------------|---------------------|------------|
| Выборы | Polls | Опросы и голосования |
| Избиратели | Participants | Участники опросов |
| Партии | Teams | Команды/группы участников |
| Жалобы | Feedback | Обратная связь и предложения |
| Сотрудники | Staff | Администраторы системы |
| Роли | Roles | Роли пользователей |
| UsersПартии | UserTeams | Связь пользователей и команд |
| UsersСотрудники | UserStaff | Связь пользователей и сотрудников |

## Переименование полей

### Polls (бывш. Выборы)
- `id_выборов` → `poll_id`
- `Название_выборов` → `poll_name`
- `Дата_начала` → `start_date`
- `Дата_окончания` → `end_date`
- `Тип_выборов` → `poll_type`
- `Статус` → `status`
- `Дата_создания` → `created_at`

### Participants (бывш. Избиратели)
- `id_избирателя` → `participant_id`
- `ФИО` → `full_name`
- `Дата_рождения` → `birth_date`
- `Адрес` → `address`
- `Дата_регистрации` → `registration_date`
- `Электронная_почта` → `email`
- `Телефон` → `phone`

### Voting (Голосование)
- `id_голоса` → `vote_id`
- `id_избирателя` → `participant_id`
- `id_выборов` → `poll_id`
- `Время_голосования` → `vote_time`
- `Месяц_голосования` → `vote_month`
- `id_адрес` → `location_id`

### Teams (бывш. Партии)
- `id_партии` → `team_id`
- `Название_партии` → `team_name`
- `Дата_регистрации` → `registration_date`
- `Статус` → `status`

### Results (Результаты_голосования)
- `id_результата` → `result_id`
- `id_выборов` → `poll_id`
- `id_партии` → `team_id`
- `Количество_голосов` → `vote_count`
- `Процент` → `percentage`
- `Дата_обработки` → `processed_at`

### Feedback (бывш. Жалобы)
- `id_жалобы` → `feedback_id`
- `id_избирателя` → `participant_id`
- `id_администратора` → `admin_id`
- `Тип_жалобы` → `feedback_type`
- `Описание` → `description`
- `Дата_жалобы` → `created_at`
- `Статус` → `status`

### Staff (бывш. Сотрудники)
- `id_администратора` → `staff_id`
- `ФИО` → `full_name`
- `Дата_рождения` → `birth_date`
- `Адрес` → `address`
- `Дата_регистрации` → `hire_date`
- `Электронная_почта` → `email`
- `Телефон` → `phone`

### Roles (Роли)
- `id_Роли` → `role_id`
- `Название_роли` → `role_name`

## Примеры использования

### Корпоративный опрос
```
Poll: "Выбор цвета нового офиса"
Teams: "Синяя команда", "Зеленая команда", "Красная команда"
Participants: Сотрудники компании
```

### Хакатон
```
Poll: "Лучший проект хакатона 2025"
Teams: Команды участников
Participants: Жюри и участники
```

### Опрос сотрудников
```
Poll: "Предпочтения по корпоративному обучению"
Options: Различные варианты курсов
Participants: Все сотрудники
```

---
**Примечание**: Вся терминология адаптирована для использования в корпоративных, учебных и развлекательных целях.
